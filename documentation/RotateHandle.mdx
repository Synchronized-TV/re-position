# RotateHandle

A simple handle for indicating that an element is rotatable. This Component
will only show when hovered. Used internally by `<TransformBox />`.

## Example Usage

<Playground>
    <div
        style={{
            position: 'relative',
            width: '200px',
            height: '200px',
            margin: 'auto',
            backgroundColor: '#f8f8f8'
        }}
    >
        <RotateHandle left top />
        <RotateHandle right top />
        <RotateHandle left bottom />
        <RotateHandle right bottom />
    </div>
</Playground>

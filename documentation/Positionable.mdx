# Positionable

`Positionable`

## Example Usage

The following is a simple example of a implementation that only provides a
single resize handle in the bottom-right hand corner. Positionable does not
render any of its own elements, so you have all the flexibility you need
to render things how you see fit.

import { Playground, PropsTable } from 'docz'
import Positionable from '../src/Positionable';

<Playground style={{
    backgroundColor: '#f0f0f0',
    height: '400px',
    overflow: 'hidden',
    position: 'relative',
}}>
    <Positionable
        resizable
        position={{
            left: '0%',
            top: '0%',
            height: '40%',
            width: '40%',
            rotation: '0deg',
        }}
    >
        {({ position, refHandlers }) => (
            <div
                ref={refHandlers.container}
                style={Object.assign({}, position, {
                    backgroundColor: "#ddd",
                    position: 'relative',
                })}
            >
                <span>Hello!</span>

                <div
                    ref={refHandlers.seResize}
                    style={{
                        position: 'absolute',
                        bottom: '-5px',
                        right: '-5px',
                        width: '10px',
                        height: '10px',
                        backgroundColor: '#f00',
                    }}
                />
            </div>
        )}
    </Positionable>
</Playground>

# TransformBox

An outline with optional handles used to indicate that an element is
Positionable. This is used by default with `<PositionableContainer />`,
but we have exposed it for any custom `<Positionable />` implementation
you may dream up.

## Example Usage

<Playground style={{ height: '300px' }}>
    <TransformBox
        // Same as what is provided to <PositionableContainer />
        //or <Positionable />.
        position={{
            left: '50%',
            top: '50%',
            width: '100px',
            height: '100px',
            rotation: '20deg',
        }}
        resizable
        rotatable

        // Any or all of the following. Generally, these come from
        // Positionable's render props (as `refHandlers`). `container`
        // is required for almost all Positionable functionality.
        //
        // container
        //
        // neRotate seRotate swRotate nwRotate
        // nResize neResize eResize seResize
        //
        // sResize swResize wResize nwResize
        refHandlers={{}}

        // Just here for the demo
        style={{ marginLeft: '-50px', marginTop: '-50px' }}
    />
</Playground>
